

  # SpriteAI: Streamlining Game Asset Creation with AI

SpriteAI is a powerful Node.js SDK designed to revolutionize the process of creating game assets, particularly for 2D games and Phaser.js projects. By leveraging advanced AI technologies, including OpenAI's DALL-E 3 and GPT-4 Vision, SpriteAI offers a suite of tools that automate and enhance various aspects of game asset creation.

## Key Features

1. **Sprite Generation**: Create custom sprite sheets for walking animations based on text descriptions.
2. **House Asset Creation**: Generate 2D house assets for game environments using AI.
3. **Background Removal**: Automatically remove specific background colors from images.
4. **Color Palette Extraction**: Analyze images to extract unique colors for palette creation.
5. **Image Processing**: Perform color replacements and manipulate image transparency.
6. **Image Encoding**: Convert images to Base64 strings for easy transmission and storage.

## Core Functionalities

- **generateSprite**: Produces sprite sheets optimized for walking animations in a Super Nintendo-inspired style.
- **generateHouseAsset**: Creates 2D house assets based on textual descriptions.
- **removeBackgroundColor**: Removes specified background colors from images, creating transparency.
- **getUniqueColors**: Extracts a list of unique colors from an image for color analysis.
- **Color Replacement**: Replaces specific colors in images with transparency or other colors.
- **encodeImage**: Converts image files into Base64 encoded strings.

## Benefits

- Rapid prototyping of game characters and environments
- Consistent style across multiple assets
- Time-saving automation of tedious asset creation tasks
- Easy integration with game engines like Phaser.js
- Flexibility in asset customization through descriptive inputs

SpriteAI aims to empower game developers and designers by simplifying the asset creation process, allowing them to focus more on game design and mechanics rather than time-consuming asset production.
